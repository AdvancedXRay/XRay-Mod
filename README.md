## Fgt X-Ray
Minecraft Forge XRay mod

## Feature:
    - Works with Forge
    - Awesome GUI
    - Able to add new ores/blocks
    - Color selector ( RGB )

Working on: Minecraft 1.7.10 and Forge 10.13.2.1291

##### Tested with:
Minecraft Forge 1.7.10 
FTB: Direwolf20 modpack
